Заготовка для мобильного приложения. Анимированное меню, часть компонентов на ReactNative, стили и цвета вынесены в отдельные конфиги

Для установки пакетов и запуска приложения

```bash
npm i && npm start
```
или

```bash
yarn && yarn start
```
